## ℹ️ О проекте

**2048 WinForms App** — это десктопная реализация популярной головоломки 2048 с расширенным функционалом.

### 🎮 Особенности:
- **Настраиваемое поле**: выберите размер игрового поля
- **Система пользователей**: ввод имени игрока
- **Таблица лидеров**: сохранение и отображение лучших результатов
- **JSON хранение**: все результаты сохраняются в файле
- **Красочный интерфейс**: каждая плитка имеет свой цвет

### 🧠 Что развивает:
- ✅ Стратегическое мышление
- ✅ Математические способности
- ✅ Пространственное планирование
- ✅ Терпение и настойчивость

## 📜 Правила игры

### Основная механика:
1. **Плитки появляются**: после каждого хода появляется новая плитка (2 или 4)
2. **Движение стрелками**: используйте ←↑→↓ для перемещения всех плиток
3. **Соединение**: одинаковые плитки при столкновении объединяются
4. **Цель**: получить плитку с числом **2048**

### Детали:
- Плитка **2** появляется с вероятностью 75%
- Плитка **4** появляется с вероятностью 25%
- Очки начисляются за каждое объединение плиток
- Игра заканчивается, когда нет доступных ходов

## 🎮 Как играть

### Начало игры:
1. Запустите приложение
2. Введите ваше **имя**
3. Выберите **размер поля** (например, 4)
4. Нажмите **"Начать игру"**

### Управление:
- **↑ Вверх** — сдвинуть все плитки вверх
- **↓ Вниз** — сдвинуть все плитки вниз
- **← Влево** — сдвинуть все плитки влево
- **→ Вправо** — сдвинуть все плитки вправо

### Стратегия:
1. Держите большие числа в углу
2. Не делайте случайных ходов
3. Стройте цепочки по возрастанию
4. Избегайте малых чисел в центре

## 🔑 Ключевые компоненты

### Map.cs:
- `Map_KeyDown()` — обработка нажатий клавиш

### DataStorage.cs:
- `Save()` — сохранение результата
- `ReadAll()` — загрузка всех результатов

## 🏆 Система рекордов

### Как работает:
1. После окончания игры результат сохраняется в `results.json`
2. Данные включают: **имя игрока** и **лучший счёт**
3. Таблица лидеров показывает топ игроков
4. Результаты сортируются по убыванию счёта
